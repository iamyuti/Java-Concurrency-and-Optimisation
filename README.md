# Java Concurrency & Optimisation

Advanced Java projects demonstrating parallel processing, functional programming, reflection, and generic type systems.

## Projects

| Project | Key Concepts |
|---------|-------------|
| **01 - Optimisation** | Functional programming, Streams, Lambdas, Function composition |
| **02 - Parallel Engine** | Multi-threading, IPC, AtomicInteger, synchronized blocks |
| **03 - Generics** | Custom generic collections, Iterator pattern, Type safety |
| **04 - Reflection** | Custom annotations, Runtime introspection, Simulation framework |
| **05 - Type Hierarchy** | Polymorphism, Inheritance, Behavioral subtyping, Test coverage |

## Technologies

- Java 17+ (Records, Pattern Matching)
- Concurrency (Threads, Atomic operations, Process pipelines)
- Functional Programming (Streams, Lambdas, Higher-order functions)
- Metaprogramming (Annotations, Reflection API)

## Run

```bash
cd 01-Optimisation-Functional
javac *.java && java Test
```
